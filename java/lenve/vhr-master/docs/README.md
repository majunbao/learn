## 项目简介

微人事是一个前后端分离的人力资源管理系统，项目采用 SpringBoot + Vue 开发。

### 技术栈

**后端技术栈**

- Spring Boot
- Spring Security
- MyBatis
- MySQL
- Redis
- RabbitMQ
- FastDFS

**前端技术栈**

- Vue
- ElementUI
- axios

### 项目地址

- GitHub: https://github.com/lenve/vhr
- Gitee: https://gitee.com/lenve/vhr

## 学习建议

建议按照文档顺序依次学习，从基础的环境搭建到高级的权限管理，循序渐进地掌握前后端分离开发的各项技能。

每个文档都包含完整的代码示例和详细的解释，适合初学者和有一定经验的开发者学习参考。
